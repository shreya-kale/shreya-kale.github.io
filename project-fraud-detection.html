LLM-POWERED FRAUDULENT JOB DETECTION

Academic / Research Project

Project Overview

This project emerged from addressing the critical problem of fraudulent job listings in the digital recruitment space, particularly during the remote work boom. The objective was to develop an advanced fraud detection model using Large Language Models (LLMs) with prompt engineering and fine-tuning techniques to identify fraudulent job postings with exceptional accuracy.

The project demanded a creative blend of natural language processing, prompt engineering, model fine-tuning, and traditional machine learning techniques to create a robust classification system that could protect job seekers from scams and fraudulent opportunities.
