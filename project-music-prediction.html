HIT SONG PREDICTION IN INDIAN POPULAR MUSIC

Academic / Personal Project

Project Overview

This project combines a passion for music with machine learning to predict whether a song would become a hit in the Indian music industry. The goal was to extract acoustic features from audio files and build predictive models that could identify patterns distinguishing hit songs from others.

Using the Spotify Web API for metadata and advanced audio processing libraries like Librosa and Aubio, this project explored the intersection of data science and art, demonstrating how machine learning can quantify musical characteristics that resonate with audiences.
